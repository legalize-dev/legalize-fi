# Maa- ja metsätalousministeriön asetus metsätalouden kannustejärjestelmän eräiden tukien laskennassa käytettävästä puukuutiometrin keskikantohinnasta

Maa- ja metsätalousministeriön päätöksen mukaisesti säädetään metsätalouden määräaikaisesta kannustejärjestelmästä annetun lain 
                    [(71/2023) 11 §:n](/akn/fi/act/statute-consolidated/2023/71#sec_11)
                     nojalla:

##### 1 § Puukuutiometrin keskikantohinnat

Metsätalouden ympäristötuen ja kulotuksen puustokorvauksen laskennassa käytetään maakuntapohjaisesti määritellyn alueen keskikantohintoja seuraavasti:

| *Maakunta* | *Keskikantohinta* |
| --- | --- |
| Uusimaa | 41,35 |
| Varsinais-Suomi | 43,31 |
| Satakunta | 43,92 |
| Kanta-Häme | 45,76 |
| Pirkanmaa | 44,36 |
| Päijät-Häme | 46,60 |
| Kymenlaakso | 43,47 |
| Etelä-Karjala | 42,96 |
| Etelä-Savo | 43,67 |
| Pohjois-Savo | 39,07 |
| Pohjois-Karjala | 36,09 |
| Keski-Suomi | 42,36 |
| Etelä-Pohjanmaa | 39,51 |
| Pohjanmaa | 35,46 |
| Keski-Pohjanmaa | 35,97 |
| Pohjois-Pohjanmaa | 31,73 |
| Kainuu | 29,36 |
| Lappi | 28,22 |

##### 2 § Voimaantulo

Tämä asetus tulee voimaan 1 päivänä tammikuuta 2024. Tätä asetusta sovelletaan sellaisiin ympäristötukihakemuksiin, jotka ovat tulleet vireille 1 päivänä tammikuuta 2024 tai sen jälkeen.
