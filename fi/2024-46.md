# Verohallinnon päätös kaivosmineraalien verotusarvoista

Verohallinto on kaivosmineraaliverolain 
                    [(314/2023) 4 §:n](/akn/fi/act/statute-consolidated/2023/314#sec_4)
                     3 momentin nojalla päättänyt:

##### 1 § Verotusarvot

Kaivosmineraalien verotusarvot ajalle 1.1.–31.12.2024 ovat:

| Kaivosmineraali / yksikkö | euroa |
| --- | --- |
| Hopea (Ag) / kilogramma | 694,07 |
| Koboltti (Co) / tonni | 31 706,46 |
| Kromi (Cr) / tonni | 3 238,22 |
| Kulta (Au) / kilogramma | 57 727,28 |
| Kupari (Cu) / tonni | 7 848,14 |
| Litium (Li) / tonni | 113 614,98 |
| Nikkeli (Ni) / tonni | 19 875,59 |
| Palladium (Pd) / kilogramma | 39 731,81 |
| Platina (Pt) / kilogramma | 28 696,53 |
| Sinkki (Zn) / tonni | 2 452,23 |
| Lyijy (Pb) / tonni | 1 976,68 |
| Uraani (U) / kilogramma | 126,13 |

##### 2 § Voimaantulo ja soveltaminen

Tämä päätös tulee voimaan 23 päivänä tammikuuta 2024. Päätöstä sovelletaan kalenterivuoden 2024 verokauteen.
