# Opetusministeriön päätös valantehneiden kielenkääntäjien tutkinnoista suoritettavista korvauksista

Opetusministeriö on vahvistanut valantehneistä kielenkääntäjistä 28 päivänä huhtikuuta 1967 annetun lain (191/67) 9§:n määräysten mukaisesti seuraavat korvaukset:

##### 1 §

1. Kaksipuolinen yleistutkinto (vieraasta/toisesta kotimaisesta kielestä kotimaiseen kieleen ja päinvastoin):

| Maksu 175 mk, josta |  |
| --- | --- |
| puheenjohtajalle | 91 mk |
| sihteerille | 60 mk |
| valtiolle | 24 mk |

2. Yksipuolinen yleistutkinto (ainoastaan vieraasta/toisesta kotimaisesta kielestä kotimaiseen kieleen taikka päinvastoin):

| Maksu 114 mk, josta |  |
| --- | --- |
| puheenjohtajalle | 55 mk |
| sihteerille | 35 mk |
| valtiolle | 24 mk |

3. Kaksipuolisesta erityistutkinnosta (lautakunnalle suoritettuna):

| Maksu 289 mk, josta |  |
| --- | --- |
| puheenjohtajalle | 84 mk |
| asiantuntijajäsenelle, joka laatii tehtävän | 121 mk |
| sihteerille | 60 mk |
| valtiolle | 24 mk |

4. Yksipuolisesta erityistutkinnosta:

| Maksu, 175 mk, josta |  |
| --- | --- |
| puheenjohtajalle | 37 mk |
| asiantuntijajäsenelle | 79 mk |
| sihteerille | 35 mk |
| valtiolle | 24 mk |

5. Maksut ja korvaukset kohdissa 1-4 mainituista tutkinnoista, jotka asianomaisen lautakunnan erityisellä suostumuksella suoritetaan muuna kuin lautakuntien julkisesti kuuluttamana aikana, maksetaan 100 prosentilla korotettuina.

##### 2 §

Tämä päätös tulee voimaan 1 päivänä toukokuuta 1987 ja tällä päätöksellä kumotaan 15 päivänä tammikuuta 1985 annettu opetusministeriön päätös (132/85) valantehneiden kielenkääntäjien tutkinnoista suoritettavista korvauksista.
