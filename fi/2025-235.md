# Maa- ja metsätalousministeriön asetus metsätalouden kannustejärjestelmän eräiden tukien laskennassa käytettävästä puukuutiometrin keskikantohinnasta

Maa- ja metsätalousministeriön päätöksen mukaisesti säädetään metsätalouden määräaikaisesta kannustejärjestelmästä annetun lain 
                    [(71/2023) 11 §:n](/akn/fi/act/statute-consolidated/2023/71#sec_11)
                     nojalla:

##### 1 § Puukuutiometrin keskikantohinnat

Metsätalouden ympäristötuen ja kulotuksen puustokorvauksen laskennassa käytetään maakuntapohjaisesti määritellyn alueen keskikantohintoja seuraavasti:

| *Maakunta* | *Keskikantohinta, euroa* |
| --- | --- |
| Uusimaa | 50,71 |
| Varsinais-Suomi | 52,33 |
| Satakunta | 53,57 |
| Kanta-Häme | 54,86 |
| Pirkanmaa | 54,14 |
| Päijät-Häme | 56,02 |
| Kymenlaakso | 53,15 |
| Etelä-Karjala | 52,57 |
| Etelä-Savo | 53,58 |
| Pohjois-Savo | 49,72 |
| Pohjois-Karjala | 46,97 |
| Keski-Suomi | 51,58 |
| Etelä-Pohjanmaa | 47,68 |
| Pohjanmaa | 42,53 |
| Keski-Pohjanmaa | 44,11 |
| Pohjois-Pohjanmaa | 40,13 |
| Kainuu | 37,69 |
| Lappi | 34,49 |

##### 2 § Voimaantulo

Tämä asetus tulee voimaan 1 päivänä kesäkuuta 2025. Tätä asetusta sovelletaan sellaisiin ympäristötukihakemuksiin, jotka ovat tulleet vireille 1 päivänä kesäkuuta 2025 tai sen jälkeen.

Tällä asetuksella kumotaan metsätalouden kannustejärjestelmän eräiden tukien laskennassa käytettävästä puukuutiometrin keskikantohinnasta annettu maa- ja metsätalousministeriön asetus 
                                [(1222/2023)](/akn/fi/act/statute-consolidated/2023/1222)
                                .
