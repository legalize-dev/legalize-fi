# Maa- ja metsätalousministeriön asetus metsätalouden ympäristötuen laskennassa käytettävästä puukuutiometrin keskikantohinnasta

Maa- ja metsätalousministeriön päätöksen mukaisesti säädetään kestävän metsätalouden määräaikaisen rahoituslain 10 §:n nojalla sellaisena kuin se on laissa 227/2016:

##### 1 § Puukuutiometrin keskikantohinnat

Metsätalouden ympäristötuen laskennassa käytetään maakuntapohjaisesti määritellyn alueen keskikantohintoja seuraavasti:

| *Maakunta* | *Keskikantohinta* |
| --- | --- |
| Uusimaa | 38,56 € |
| Varsinais-Suomi | 41,05 € |
| Satakunta | 41,26 € |
| Kanta-Häme | 43,20 € |
| Pirkanmaa | 41,19 € |
| Päijät-Häme | 43,72 € |
| Kymenlaakso | 40,64 € |
| Etelä-Karjala | 39,86 € |
| Etelä-Savo | 40,36 € |
| Pohjois-Savo | 35,99 € |
| Pohjois-Karjala | 33,14 € |
| Keski-Suomi | 39,42 € |
| Etelä-Pohjanmaa | 36,97 € |
| Pohjanmaa | 33,16 € |
| Keski-Pohjanmaa | 33,45 € |
| Pohjois-Pohjanmaa | 29,90 € |
| Kainuu | 27,65 € |
| Lappi | 27,12 € |

##### 2 § Voimaantulo

Tämä asetus tulee voimaan 1 päivänä tammikuuta 2023. Tätä asetusta sovelletaan sellaisiin ympäristötukihakemuksiin, jotka ovat tulleet vireille 1 päivänä tammikuuta 2023 tai sen jälkeen.
