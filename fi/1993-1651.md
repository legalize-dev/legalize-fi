# Sosiaali- ja terveysministeriön päätös vakuutustarkastuksen maksullisista suoritteista

Sosiaali- ja terveysministeriö on 21 päivänä helmikuuta 1992 annetun valtion maksuperustelain (150/92) 8 §:n ja vakuutusyhtiön toimilupamaksusta 22 päivänä joulukuuta 1993 annetun lain (1421/93) 3 §:n nojalla päättänyt:

##### 1 §

Sosiaali- ja terveysministeriö perii suomalaisia vakuutusyhtiöitä, ensivakuutusliikettä Suomessa harjoittavia ulkomaisia vakuutusyrityksiä, vakuutusyhdistyksiä, vakuutuskassoja, eläkesäätiöitä ja vakuutuksenvälittäjiä sekä liikennevakuutuskeskusta, potilasvakuutusyhdistystä ja vakuutuskassojen yhdistystä koskevista vakuutustarkastuksen julkisoikeudellisista suoritteista tämän päätöksen 2 §:n mukaiset kiinteät maksut. Sanottuja maksuja ei kuitenkaan peritä lakisääteisen tapaturmavakuutuksen ehtojen vahvistamista koskevista päätöksistä eikä lakisääteisen työeläkevakuutuksen ehtojen ja perusteiden vahvistamista koskevista päätöksistä lukuunottamatta vakuutusmaksun ja vastuuvelan laskuperusteiden vahvistamispäätöksiä.

##### 2 §

Vakuutustarkastuksen julkisoikeudellisista suoritteista peritään seuraavat maksut:

| 1) | vakuutusyhtiön toimilupaa koskeva päätös: |  |
| --- | --- | --- |
|  | uuden toimiluvan antamista koskeva päätös | 30 000 mk |
|  | voimassa olevan toimiluvan muuttamista koskeva päätös | 15 000 mk |
| 2) | yhtiöjärjestyksen ja yhdistysjärjestyksen vahvistamista tai muuttamista koskeva päätös | 2 000 mk |
| 3) | sääntöjen vahvistamista tai muuttamista koskeva päätös: |  |
|  | eläkesäätiöiden, eläkekassojen, vakuutuskassojen yhdistyksen ja potilasvakuutusyhdistyksen osalta | 2 000 mk |
|  | muiden vakuutuskassojen kuin eläkekassojen osalta | 1 000 mk |
| 4) | sulautumista, jakautumista, yhteisömuodon muuttamista, vakuutuskannan luovuttamista ja vastuun siirtämistä koskeva päätös: |  |
|  | vakuutusyhtiöiden osalta | 10 000 mk |
|  | vakuutusyhdistysten, eläkesäätiöiden ja eläkekassojen osalta | 5 000 mk |
|  | muiden vakuutuskassojen kuin eläkekassojen osalta | 2 500 mk |
| 5) | vakuutusteknisten perusteiden vahvistamista koskeva päätös: |  |
|  | muiden vakuutuskassojen kuin eläkekassojen osalta | 1 000 mk |
|  | muiden 1 §:ssä tarkoitettujen vakuutus- ja eläkelaitosten osalta | 2 000 mk |
| 6) | vakuutuksenvälittäjän rekisteröintiä koskeva päätös | 1 000 mk |
| 7) | rekisteriote ja todistus | 200 mk |
| 8) | muu hakijan pyynnöstä annettava lupa ja päätös | 2 000 mk. |

##### 3 §

Tämä päätös tulee voimaan 1 päivänä tammikuuta 1994.
